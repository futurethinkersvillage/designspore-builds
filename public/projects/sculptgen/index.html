How it works

Each sculpture starts as a mathematical form field, then gets deconstructed region by region into flat parts using real construction techniques — stacked layers, radial ribs with slotted hubs, interlocked egg-crate grids, and 3D-printed connectors. Every part you see is a real cut piece: the 3D view is built from the same polygons as the downloadable cut sheets.
